Les nuages

1. altocumulus

Ce type de nuage possède les espèces suivantes :

  • lenticularis
  • stratiformis
  • castellanus
  • flocus

2. cirrus

Ce type de nuage possède les espèces suivantes :

  • flocus
  • castellanus
  • spissatus
  • uncinus
  • fibratus

3. cirrocumulus

Ce type de nuage possède les espèces suivantes :

  • lenticularis
  • stratiformis
  • flocus
  • castellanus

4. cumulus

Ce type de nuage possède les espèces suivantes :

  • fractus
  • humilis
  • mediocris
  • congestus

5. strato-cumulus

Ce type de nuage possède les espèces suivantes :

  • stratiformis
  • lenticularis
  • castellanus

6. cumulonimbus

Ce type de nuage possède les espèces suivantes :

  • calvus
  • capillatus
  • mammatus
